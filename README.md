## ✨ Overview

A **production-ready fullstack Todo application** built with modern technologies and deployed using a complete **CI/CD pipeline**.

This project demonstrates:

* 🐳 Docker containerization
* 🔄 Jenkins CI/CD automation
* ☁️ AWS cloud deployment
* 🧠 Clean backend architecture

---

## 🧩 Features

* ✅ Create, update, delete tasks
* ✅ REST API integration
* ✅ MongoDB database connection
* ✅ Dockerized fullstack app
* ✅ CI/CD pipeline automation
* ✅ AWS deployment ready

---

## 🧠 Tech Stack

| Layer    | Technology          |
| -------- | ------------------- |
| Frontend | React.js, Axios     |
| Backend  | Node.js, Express.js |
| Database | MongoDB             |
| DevOps   | Docker, Jenkins     |
| Cloud    | AWS EC2             |

---

## 🏗️ Architecture

```text
User → Nginx (Reverse Proxy)
        ↓
   Frontend (React Build)
        ↓
   Backend API (Node.js)
        ↓
   MongoDB Database
```

---

## ⚙️ Run Locally

```bash
docker compose up -d --build
```

---

## 🌐 Access Application

* Frontend → http://localhost:5000
* Backend → http://localhost:3500/api/tasks
